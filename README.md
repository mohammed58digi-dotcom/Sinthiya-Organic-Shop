## 📁 ফাইল কাঠামো

```
sinthiya-organic/
├── index.html       ← মূল ওয়েবসাইট (কাস্টমার দেখবে)
├── admin.html       ← অ্যাডমিন প্যানেল (শুধু আপনি ব্যবহার করবেন)
├── products.js      ← প্রোডাক্ট ডেটা

```

## 🚀 GitHub Pages-এ লাইভ করার পদ্ধতি

### ধাপ ১: GitHub Account
- https://github.com এ যান
- Sign up করুন (ফ্রি)

### ধাপ ২: নতুন Repository তৈরি
- New Repository ক্লিক করুন
- Repository name: `sinthiya-organic` (বা যেকোনো নাম)
- Public রাখুন
- Create Repository ক্লিক করুন

### ধাপ ৩: ফাইল আপলোড
- "uploading an existing file" লিংকে ক্লিক করুন
- সব ফাইল ড্র্যাগ করে আপলোড করুন
- "Commit changes" ক্লিক করুন

### ধাপ ৪: GitHub Pages চালু করুন
- Settings → Pages
- Branch: main, Folder: / (root)
- Save ক্লিক করুন

### ধাপ ৫: আপনার ওয়েবসাইট লাইভ!
- URL হবে: `https://আপনার-username.github.io/sinthiya-organic/`
- Admin Panel: `https://আপনার-username.github.io/sinthiya-organic/admin.html`
